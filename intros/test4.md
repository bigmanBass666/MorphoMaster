# 🎓 MorphoMaster - 英语词形变化学习系统

## 📋 项目概述

**MorphoMaster** 是一款专注于英语词形变化（Morphology）学习的交互式Web应用。项目旨在帮助学习者系统掌握英语名词复数形式和动词时态变化，通过即时反馈、进度追踪和沉浸式体验，让词形变化学习变得高效而有趣。

---

## ✨ 核心亮点

### 🎯 1. 双模式学习系统

- **名词模式** 📚：练习不规则名词复数形式（如 analysis → analyses, child → children）
- **动词模式** 🔄：同时练习过去式和过去分词（如 go → went → gone）
- **智能切换**：根据单词类型自动显示对应的输入界面，无需手动切换

### 🎨 2. 精美的用户界面设计

- **现代化UI**：采用卡片式设计，圆角阴影，视觉层次清晰
- **响应式布局**：完美适配桌面端和移动端，最大宽度600px的居中容器
- **视觉反馈系统**：
  - ✅ 正确答案：绿色背景 + 对勾图标
  - ❌ 错误答案：红色背景 + 错误提示 + 正确答案显示
  - 📊 实时进度条：直观展示学习进度

### 🌓 3. 主题切换功能

- **亮色/暗色模式**：一键切换，保护用户眼睛
- **持久化存储**：主题偏好自动保存到本地
- **智能配色**：暗色模式下自动调整彩带颜色

### 📈 4. 智能进度管理系统

- **本地存储持久化**：使用 `localStorage` 自动保存学习进度
- **断点续学**：刷新页面后自动恢复上次学习位置
- **进度可视化**：实时显示 "当前/总数" 和进度条百分比
- **完成庆祝**：完成所有单词后触发彩带动画效果 🎉

### ⌨️ 5. 键盘快捷键支持

- `Enter`：提交答案
- `Alt + H`：上一个单词
- `Alt + L`：下一个单词
- `Alt + T`：切换主题
- **无障碍设计**：支持纯键盘操作，提升学习效率

### 🎊 6. 完成庆祝动画

- **动态导入**：使用 ES6 动态导入 `confetti` 库，按需加载
- **双色彩带**：从屏幕两侧同时发射，持续10秒
- **主题适配**：根据当前主题自动调整彩带颜色

### 🔍 7. 智能输入验证系统

- **输入规范化**：
  - 自动转换为小写
  - 处理特殊字符和多余空格
  - 保留斜杠 `/` 以支持音标输入
- **即时反馈**：
  - 实时验证答案正确性
  - 错误时自动选中输入框，便于修改
  - 显示正确答案和音标

### 📚 8. 丰富的单词数据库

- **数据格式**：JSON格式，易于扩展和维护
- **完整信息**：包含原词、音标（IPA）、词形变化、中文释义
- **分类清晰**：名词和动词分别处理，数据结构统一

## 🏗️ 技术架构

### 📦 模块化设计

项目采用 ES6 模块化架构，代码组织清晰，职责分明：

```
js/
├── main.js          # 应用入口，初始化与事件绑定
├── state.js         # 全局状态管理
├── config.js        # 配置管理
├── progress.js      # 进度管理模块
├── theme.js         # 主题切换模块
└── modules/
    ├── word.js      # 单词显示逻辑
    ├── answer.js    # 答案检查入口
    ├── validation.js # 输入验证核心
    ├── progression.js # 答题流程控制
    ├── input.js     # 输入框管理
    ├── navigation.js # 导航控制
    └── confetti.js  # 彩带动画
```

### 🎯 核心设计模式

#### 1. **状态管理模式**

- 集中式状态管理（`state.js`）
- 单一数据源，避免状态混乱
- DOM元素统一缓存，提升性能

#### 2. **职责分离原则**

- **word.js**：负责单词显示和切换
- **validation.js**：纯函数验证逻辑，易于测试
- **progression.js**：处理答题后的流程控制
- **input.js**：管理输入框状态和焦点

#### 3. **事件驱动架构**

- 统一的事件监听器管理
- 键盘事件和鼠标事件分离处理
- 支持事件委托，性能优化

### 💾 数据持久化策略

- **localStorage API**：存储学习进度和主题偏好
- **错误处理**：完善的异常捕获和用户提示
- **数据校验**：加载前检查数据有效性

### 🎨 CSS架构设计

- **CSS变量系统**：统一管理颜色和尺寸（`vars.css`）
- **工具类**：复用性高的工具类（`utils.css`）
- **响应式设计**：移动优先的设计理念
- **动画效果**：流畅的过渡动画和错误提示动画

## 📱 功能演示流程

1. **启动应用** 🚀
   - 自动加载单词数据
   - 恢复上次学习进度
   - 应用保存的主题设置

2. **学习流程** 📖
   - 显示当前单词、音标和中文释义
   - 根据单词类型显示对应输入框
   - 用户输入答案并提交

3. **答案验证** ✅
   - 实时验证答案正确性
   - 正确：绿色高亮，自动进入下一题
   - 错误：红色高亮，显示正确答案

4. **进度更新** 📊
   - 实时更新进度条和计数
   - 自动保存进度到本地存储

5. **完成庆祝** 🎉
   - 完成所有单词后触发彩带动画
   - 自动重置进度，可重新开始
